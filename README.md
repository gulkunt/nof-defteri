Not Defteri Uygulaması

Bu proje, kullanıcıların notlarını oluşturmasını, görüntülemesini ve yönetmesini sağlayan basit bir not defteri uygulamasıdır. Proje, temel programlama ve kullanıcı arayüzü geliştirme becerilerini geliştirmek amacıyla hazırlanmıştır.

##Özellikler
Yeni not ekleme
Notları görüntüleme
Notları düzenleme
Notları silme
Kullanıcı dostu arayüz
Kullanılan Teknolojiler
##HTML
CSS
JavaScript
Projenin Amacı

Bu proje, temel web geliştirme becerilerimi geliştirmek ve kullanıcı arayüzü tasarımı konusunda deneyim kazanmak amacıyla hazırlanmıştır.
